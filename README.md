# Welcome to My Bouncing Box
***

## Task
The challenge here was to use html and js to make a box that bounce at the border.

## Description
The program will make a box and take the max left and top of the window and will detect when the box will touch the border and will make the box bounce on it.

## Installation
There was nothing to install for this project.

## Usage
Just open a window to make it work.
